# ✈️ Airplane Management System
A simple and responsive Airplane Management System dashboard developed using HTML, CSS, and Bootstrap 5. This project provides a modern user interface for managing flights, tickets, departures, and arrivals.

## 🚀 Features
- Responsive Dashboard
- Sidebar Navigation
- Bootstrap 5 UI Components
- Ticket Management Page
- Departures Page
- Arrivals Page
- Flight Statistics Cards
- Flight Schedule Tables
- Banner Images
- Bootstrap Icons
- Clean and User-Friendly Design

## 🛠️ Technologies Used
- HTML5
- CSS3
- Bootstrap 5
- Bootstrap Icons
